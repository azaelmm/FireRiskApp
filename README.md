# 🔥 FireRiskNet  
**Predicción del riesgo de incendios forestales mediante Redes Neuronales Artificiales (NNA)**  
Proyecto desarrollado en Python con FastAPI y un frontend interactivo en React + TypeScript + Leaflet.

---

## 🌲 Descripción del proyecto  
**FireRiskNet** es un sistema de predicción del riesgo de incendios forestales basado en datos ambientales como:

- Temperatura  
- Humedad del aire  
- Humedad del suelo  
- Velocidad y dirección del viento  
- Índices de sequía  
- Variables climáticas adicionales  

El objetivo es ofrecer una herramienta accesible que permita visualizar el riesgo de incendio en un mapa interactivo y realizar predicciones en tiempo real mediante un modelo de **Deep Learning** entrenado con datos históricos.

---

## 🧠 Tecnologías utilizadas

### 🔧 Backend (Python)
- **FastAPI** — API rápida y moderna para servir el modelo  
- **TensorFlow / PyTorch / Keras** — para entrenar la NNA  
- **Pandas / NumPy** — procesamiento de datos  
- **Scikit‑learn** — preprocesamiento y métricas  
- **Uvicorn** — servidor ASGI  

### 🎨 Frontend (TypeScript)
- **React + Vite** — interfaz web moderna  
- **Leaflet** — mapas interactivos  
- **React‑Leaflet** — integración con React  
- **OpenStreetMap** — mapas gratuitos
